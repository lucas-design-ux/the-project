# Master Template - The Blog Ecosystem

This project serves as the foundational scaffold for the "Master Template" ecosystem. It is built with:

- **Next.js 14+** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Atomic Design** component structure

## Getting Started

1.  Iinstall dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app`: Application routes and layouts.
- `/components`: Atomic Design components (`atoms`, `molecules`, `organisms`, `templates`).
- `/config`: Global configuration (e.g., themes).
- `/lib`: Utility functions and libraries.
- `/hooks`: Custom React hooks.
- `/types`: TypeScript type definitions.

## Scripts

- `npm run dev`: Start development server.
- `npm run build`: Build for production.
- `npm run start`: Start production server.
- `npm run lint`: Run ESLint.
- `npm run format`: Format code with Prettier.
